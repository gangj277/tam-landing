# 퀴즈 퍼널 아이디어 아카이브

> 부모 유입을 위한 퀴즈 리드 마그넷 전략. First principle 기반 설계.

---

## 설계 원칙

### 한국 부모가 퀴즈를 완료하는 이유
1. **블라인드 스팟 공포** — "나는 우리 아이를 잘 알고 있을까?"
2. **정보력 과시** — 먼저 발견하고 공유하면 사회적 자본
3. **시계 불안** — 10-14세는 골든타임, "지금 안 하면 늦는다"
4. **숨겨진 잠재력** — 모든 부모는 아이가 특별하다고 믿고 싶다
5. **새로운 기준** — 기존에 없던 프레임을 제시하면 무시할 수 없다

### 최적 구조
- **문항 수**: 7~9개 (신뢰도와 완료율의 균형)
- **소요 시간**: 2~3분
- **중간 리빌**: 4번째 질문 후 "지금까지의 패턴은..." 으로 이탈 방지
- **결과 포맷**: 유형 이름(2~4음절) + 시각적 카드 (카톡 공유용 스크린샷)
- **결과 원칙**: 어떤 결과든 긍정적 프레이밍 → 하지만 행동 유도

### 결과 페이지 퍼널
```
결과 카드 (3초 흡수) → 상세 분석 (스크롤) → 카카오톡 공유 버튼 →
탐 소개 (소프트셀) → 연락처 수집 → "아이와 함께 다시 해보기" 유도
```

## 퀴즈 7개 상세

---

### 1. 🔮 우리 아이 AI 시대 생존 유형 테스트

**감정 트리거**: AI 불안 + 유형화 욕구
**불안 수준**: 높음 / **공유성**: 높음

**8문항, 시나리오 기반**
- Q1-Q2: 새로운 기술에 대한 반응 (관찰자 vs 뛰어드는 자 vs 질문자 vs 만드는 자)
- Q3-Q4: 불확실성/모호함 대처 방식
- Q5-Q6: 정보 소비 vs 생산 패턴
- Q7-Q8: 미래 상상 방식

**6가지 유형 결과**
| 유형 | 설명 |
|------|------|
| 🧭 방향 설계자 | 원하는 게 분명, 도구가 필요한 아이 |
| 🔍 질문 발굴자 | AI가 못하는 질문을 던지는 아이 |
| 🎨 융합 창작자 | 분야를 넘나들며 조합하는 아이 |
| 🤝 연결 중재자 | 사람과 기술을 연결하는 높은 EQ |
| 🧪 실험 탐험가 | 해봐야 아는 아이, 실패 허용이 필요 |
| 📖 깊이 전문가 | 넓이보다 깊이, 집중 멘토링 필요 |

**탐 연결**: "[유형] 아이에게 가장 필요한 경험은 [X]입니다. 탐은 매일 아이의 유형에 맞춘 세계를 열어줍니다."

**소요**: 2분 30초

### 2. 👁️ 엄마(아빠)는 모르는 우리 아이의 숨겨진 세계

**감정 트리거**: 블라인드 스팟 공포 + 자기 평가
**불안 수준**: 중상 / **공유성**: 매우 높음

**9문항, 이중 응답 형식**
각 질문에 두 번 답변: ① 아이라면 이렇게 답할 것 같다 ② 부모인 나의 바람

- "아이가 자유시간에 하는 일은?" (아이 예측 vs 부모 희망)
- "아이가 가장 걱정하는 것은?"
- "아이 눈이 반짝이는 주제는?"
- "아이가 자신이 못한다고 생각하는 것은?"
- "일주일 동안 뭐든 배울 수 있다면?"
- "아이가 가장 존경하는 사람은?"

**결과: 이해도 지도**
- 🟢 잘 아는 영역 — 검증
- 🟡 흐릿한 영역 — 호기심
- 🔴 미지의 영역 — 긴급

**바이럴 포인트**: "나 이해도 72%래" — 엄마 그룹에서 자연스러운 겸손 자랑/자기비하 유머

**탐 연결**: "아이의 '미지의 영역'을 발견하는 가장 좋은 방법은 아이 스스로 다양한 세계를 경험하게 하는 것입니다."

**소요**: 3분

### 3. ⏰ 2030년, 우리 아이 준비도 체크

**감정 트리거**: 시계 불안 + 사회적 비교
**불안 수준**: 높음 / **공유성**: 중상

**8문항, 벤치마크 비교 형식**
각 답변 후 "다른 부모님들의 응답" 실시간 표시 → 사회적 비교 엔진 가동

- "AI로 학교 프로젝트를 완성해야 한다면?"
- "AI가 만든 콘텐츠를 알아챌 수 있을까?"
- "AI가 쉽게 요약하는 주제에서 자기만의 관점을 말할 수 있을까?"
- "완전히 다른 배경의 사람과 협업해야 한다면?"
- "꿈꾸는 직업이 AI로 바뀐다고 하면?"

**결과: 4사분면 맵**
| | AI 활용력 높음 | AI 활용력 낮음 |
|---|---|---|
| 자기이해 높음 | 🌟 미래 주도형 | 🎯 방향은 있는데 도구가 없는 아이 |
| 자기이해 낮음 | ⚡ 도구는 있는데 방향이 없는 아이 | 🌱 아직 탐색 중인 아이 |

+ 0~100점 준비도 점수

**탐 연결**: 각 사분면마다 맞춤 처방 → 모든 길이 탐으로 연결

**소요**: 3분

### 4. 🧬 우리 아이 몰입 DNA 분석

**감정 트리거**: 숨겨진 잠재력 + 경이감
**불안 수준**: 낮음 / **공유성**: 매우 높음

⭐ **첫 번째로 런칭 추천** — 가장 부담 없고 공유성 최고

**7문항, 패턴 인식 형식**
- Q1: "시간 가는 줄 모르는 활동 3개 고르기" (12개 그리드)
- Q2: "깊이 빠질 때의 행동 패턴"
- Q3: "호기심의 시작점" (질문 / 감정 / 본 것 / 만난 사람 / 문제)
- Q4: "어려움을 만났을 때"
- Q5: "에너지가 올라가는 상황" (사회적 vs 혼자 / 만들기 vs 분석)
- Q6: 중간 리빌 — "여기까지의 공통 실마리는 [X]인 것 같아요. 맞나요?"
- Q7: "모든 판단을 제거하면, 하루 종일 뭘 할까?"

**5가지 몰입 DNA**
| DNA | 핵심 동력 | 몰입 트리거 |
|-----|----------|------------|
| 🔬 탐구형 | "왜?"를 이해하고 싶다 | 새로운 정보 + 자율성 |
| 🎭 표현형 | 보여주고 들려주고 싶다 | 관객 + 자유 |
| 🏗️ 구축형 | 뭔가 실체를 만들고 싶다 | 산출물 + 반복 개선 |
| 🌐 연결형 | 사람/아이디어를 잇고 싶다 | 사회적 맥락 + 다양성 |
| ⚖️ 해결형 | 부서진 걸 고치고 싶다 | 명확한 문제 + 자원 |

**바이럴 포인트**: "우리 아이 구축형 DNA래! 진짜 맞는 것 같아!" — 새로운 언어를 부여하는 것 자체가 공유 동기

**탐 연결**: "[DNA 유형] 아이는 [미션 유형]에서 가장 강한 몰입을 경험합니다."

**소요**: 2분 30초

### 5. 💬 AI에게 물어봤습니다: "당신의 아이를 설명해주세요"

**감정 트리거**: 메타적 불안 + 통찰
**불안 수준**: 중상 / **공유성**: 매우 높음

**8문항, AI 분석 시뮬레이션 형식**
UI가 마치 AI가 실시간으로 분석하는 것처럼 디자인 (로딩 애니메이션)

- "아이가 유튜브/인터넷에서 가장 많이 검색하는 것은?"
- "스마트폰 사용 패턴은?"
- "AI에 대해 놀라운 질문을 한 적 있나?"
- "AI가 만든 콘텐츠에 대한 반응은?"
- "AI와 대화할 수 있다면 어떤 주제?"
- "인간이 만든 것과 AI가 만든 것을 구별할 수 있나?"

**Q6 이후 충격 리빌**: "AI가 본 우리 아이의 프로필" 자동 생성

**결과: 이중 초상화**
- **왼쪽**: AI가 보는 아이 — 데이터 기반, 의도적으로 환원적
- **오른쪽**: AI가 못 보는 아이 — 부모의 감성적/맥락적 답변 기반, 풍부하고 인간적

"이 차이를 이해하는 아이가 AI 시대의 진짜 주인공입니다."

**탐 연결**: "AI가 보는 것과 보지 못하는 것의 차이를 이해하려면, 아이가 직접 AI와 함께 다양한 세계를 경험해야 합니다."

**소요**: 3분

### 6. 🎯 우리 아이는 어떤 세계에 살고 있을까?

**감정 트리거**: 경이감 + 연결 욕구
**불안 수준**: 낮음 / **공유성**: 매우 높음

⭐ **탐의 제품 컨셉과 가장 직결** — "다양한 세계" 메시지와 1:1 매칭

**7문항, 스토리/시나리오 형식**
미니 이야기 제시 → 아이의 반응 예측

- "학교에 어제 없던 신비한 문이 생겼다. 안에서 소리가 들린다. 아이는?"
- "2050년에서 온 사람이 도움을 요청하는 메시지를 보냈다. 아이의 반응은?"
- "학교에서 딱 한 가지를 바꿀 수 있다면?"
- "배운 적 없는 언어를 갑자기 알게 됐다면 제일 먼저?"
- "자기가 좋아하는 것에 반대하는 사람을 만나면?"
- "자기 자신을 표현하는 무언가를 만든다면 어떤 형태?"
- "규칙도 숙제도 제한도 없는 하루. 어떻게 보낼까?"

**결과: 세계 지도 (4개 영역)**
| 영역 | 아키타입 | 동력 |
|------|---------|------|
| 🏔️ 발견의 세계 | 탐험가 | "저 너머에 뭐가 있지?" |
| 🎪 표현의 세계 | 예술가 | "내가 뭘 만들 수 있지?" |
| 🏛️ 질서의 세계 | 건축가 | "이건 어떻게 작동하지?" |
| 🌿 관계의 세계 | 치유자 | "누가 도움이 필요하지?" |

홈 베이스 + 보조 영역으로 표시. 모든 아이가 모든 세계를 방문하되, 거점이 있다.

**바이럴 포인트**: 세계 지도가 시각적으로 아름다움 + 이야기 형식 질문이 대화 소재 ("진짜 물어봤더니 그렇게 답하더라!")

**탐 연결**: "아이의 세계를 넓혀주세요. 탐은 매일 아이가 아직 가보지 않은 세계의 문을 열어줍니다."

**소요**: 2분 30초

### 7. 📊 학원 5개 vs. 경험 1개: 우리 아이에게 진짜 필요한 건?

**감정 트리거**: 죄책감 + 해방감
**불안 수준**: 높음 / **공유성**: 높음 (양극화 유발)

⚠️ **가장 도발적** — 런칭 초기에는 비추, 브랜드 확립 후 사용

**8문항, 현실 직면 형식**
현재 스케줄 팩트 체크 + 성찰 질문 교차

- "일주일에 구조화된 학습 활동 몇 개?" (1~2 / 3~4 / 5~6 / 7+)
- "지난 한 달, 아이가 순수하게 하고 싶어서 한 활동은 몇 번?"
- "아이가 자기 관심사를 자기 말로 설명할 수 있나?"
- "아이가 마지막으로 진심으로 심심했던 게 언제?" (심심함 = 기회)
- "내일 학원을 다 끊으면 아이는 시간을 뭘 하며 보낼까?"
- "학원에서 안 온, 아이가 스스로 발견한 것이 있나?"
- "모든 활동에서 딱 하나의 결과를 보장받는다면?"
- "솔직히: 아이의 스케줄은 아이의 미래를 위한 건가, 부모의 안심을 위한 건가?"

**Q8이 킬러 질문** — 따뜻하지만 정면 도전

**결과: 3축 스펙트럼**
1. 구조 ←→ 자유
2. 외부동기 ←→ 내부동기
3. 넓이 ←→ 깊이

**4가지 프로필**
- ⚖️ 균형 잡힌 탐험가 — 잘하고 있다
- 📚 과부하 학습자 — 숨 쉴 공간 필요
- 🎮 방향 없는 자유인 — 가이디드 탐험 필요
- 🔍 숨겨진 열정 소유자 — 뭔가 있는데 공간이 없다

**탐 연결**: "학원은 '정해진 답'을 가르칩니다. 탐은 '자기만의 질문'을 찾게 합니다."

**소요**: 3분
