# 학부모 설문조사 기획서 v2

> **목적**: 제품 가설의 간접 검증 + 베타 유저 모집  
> **대상**: 초등학생 자녀를 둔 학부모  
> **예상 소요시간**: 4분  
> **도구**: 구글폼  
> **핵심 원칙**: 우리가 원하는 답을 유도하지 않는다. 부모의 자연스러운 우선순위에서 신호를 읽는다.

---

## 설계 원칙

1. **간접 측정**: "X가 중요합니까?"라고 묻지 않는다. 여러 선택지 중에서 순위를 매기거나 포인트를 배분하게 해서, 자연스럽게 우선순위가 드러나도록 한다.
2. **행동 기반 질문**: 태도("~하고 싶다")보다 행동("실제로 ~하고 있다")을 먼저 묻는다. 행동은 거짓말하기 어렵다.
3. **깔때기 구조**: 넓은 질문 → 좁은 질문 → 솔루션 노출. 앞 섹션에서 우리 제품을 절대 언급하지 않는다.
4. **뒤로가기 차단**: 구글폼 설정에서 "이전 섹션으로 돌아가기" 비활성화. 솔루션 노출 후 앞 답변을 수정하지 못하게.
5. **편향 방지**: 선택지 순서를 랜덤화(구글폼 "선택지 순서 섞기" 옵션 활성화). 부정 선택지("딱히 없다", "잘 모르겠다")를 반드시 포함.

## 검증 가설과 측정 방법

| ID | 가설 | 측정 방법 | 검증 기준 |
|----|------|----------|----------|
| H1 | 부모는 "아이의 자기이해/흥미발견"을 교육의 주요 우선순위로 본다 | Q3 순위 매기기에서 해당 항목의 평균 순위 | 6개 항목 중 평균 3위 이내 |
| H2 | 부모는 아이가 자기 흥미/강점을 충분히 파악하고 있지 않다고 느낀다 | Q5 행동 기반 질문 + Q6 간접 측정 | Q5에서 주 2시간 이하 비율 60%↑, Q6에서 "잘 모르겠다" 40%↑ |
| H3 | 현재 이 니즈를 해결하는 마땅한 수단이 없다 | Q7 현재 방법 + Q8 포인트 배분 | Q7 "특별히 없음" 30%↑ 또는 Q8에서 해당 영역 배분 15점 이하 |
| H4 | AI 기반 솔루션에 대한 수용도가 있다 | Q10 사용 의향 + Q11 지불의향 | Q10 4~5점 50%↑, Q11 유료 의향 30%↑ |

## 섹션 2: 교육 우선순위 (H1 검증)

> 섹션 설명: "자녀 교육에 대한 생각을 여쭤보겠습니다."

### Q3. 아이 교육에서 중요하다고 생각하는 것의 순위를 매겨주세요 (1위~6위)
- **유형**: 순위 매기기 (구글폼 "순위" 유형, 또는 격자형으로 1~6위 배정)
- **필수**: O
- **항목** (순서 랜덤):
  - 학업 성적/성취
  - 자기 흥미와 강점 발견
  - 사회성/또래 관계
  - 체력/건강 관리
  - 창의성/문제해결력
  - 바른 인성/생활습관
- **의도**: "흥미와 강점 발견"의 자연 순위를 확인. 직접 물으면 모두 중요하다고 하지만, 강제 순위에서 실제 우선순위가 드러남.
- **검증**: H1 — "자기 흥미와 강점 발견"이 평균 3위 이내에 들면 가설 지지.
- **분석 팁**: 학년별로 순위 차이가 있는지도 크로스탭 분석. 저학년(1~3) vs 고학년(4~6) 비교.

### Q4. 현재 자녀 교육비를 100이라 할 때, 각 영역에 얼마나 쓰고 계신가요? (합계 100)
- **유형**: 단답형 숫자 6개 (격자형 또는 개별 입력)
- **필수**: O
- **항목** (순서 랜덤):
  - 교과 학습 (학원, 과외, 학습지 등)
  - 예체능 (음악, 미술, 체육 등)
  - 체험/캠프/여행
  - 독서/교양
  - 진로/적성 탐색
  - 기타
- **의도**: Q3이 "생각하는 중요도"라면, Q4는 "실제 돈을 어디에 쓰는가". 이 둘의 괴리가 핵심 인사이트. 진로/적성 탐색에 "중요하다고는 생각하지만 돈은 안 쓴다"면 → 적절한 수단이 없어서(H3 보강).
- **검증**: H1 보강 — Q3에서 높은 순위 vs Q4에서 실제 지출 비교.
- **구현 참고**: 구글폼에서 합계 100 강제가 어려우면, "대략적으로 합이 100이 되도록 적어주세요"로 안내하고, 분석 시 정규화.

## 섹션 3: 아이의 현재 상태 (H2 검증)

> 섹션 설명: "자녀의 일상에 대해 여쭤보겠습니다."

### Q5. 아이가 일주일에 "스스로 선택한 활동"에 보내는 시간은 대략 얼마인가요?
- **유형**: 단일선택
- **필수**: O
- **선택지**:
  - 거의 없음 (학원/숙제 외 자유시간이 부족)
  - 주 1~2시간 정도
  - 주 3~5시간 정도
  - 주 5시간 이상
  - 잘 모르겠다
- **의도**: 행동 기반으로 H2 간접 측정. 스스로 선택한 활동 시간이 적다면, 자기 흥미를 탐색할 기회 자체가 부족한 것. 태도가 아니라 사실을 묻는 질문.
- **검증**: H2 — "거의 없음" + "1~2시간" 합계 60% 이상이면 가설 지지.

### Q6. 아이가 요즘 가장 관심 있어하는 것을 한 가지만 꼽는다면?
- **유형**: 단일선택
- **필수**: O
- **선택지**:
  - 특정 과목 (수학, 과학, 역사 등)
  - 게임/유튜브 등 디지털 콘텐츠
  - 스포츠/운동
  - 예술 (그림, 음악, 만들기 등)
  - 동물/자연/과학 탐구
  - 또래 친구와 노는 것
  - 솔직히 잘 모르겠다
  - 기타 (직접 입력)
- **의도**: "솔직히 잘 모르겠다"의 비율이 H2의 핵심 지표. 부모가 아이의 관심사를 구체적으로 모른다 = 아이의 자기이해가 부모에게 전달되지 않고 있다는 증거. 직접 "아이가 자기를 잘 모르나요?"라고 묻는 것보다 훨씬 솔직한 답이 나옴.
- **검증**: H2 — "잘 모르겠다" 비율 30% 이상이면 강한 신호. 40% 이상이면 매우 강한 신호.

## 섹션 4: 현재 수단과 불만 (H3 검증)

> 섹션 설명: "자녀의 진로·흥미 탐색에 대해 여쭤보겠습니다."

### Q7. 아이의 흥미/강점을 파악하기 위해 현재 하고 있는 것은? (복수선택)
- **유형**: 체크박스
- **필수**: O
- **선택지** (순서 랜덤):
  - 적성검사/성향검사 (종이 또는 온라인)
  - 다양한 체험학습/캠프 참여
  - 학원/과외 선생님과 상담
  - 학교 담임 선생님과 상담
  - 관련 책/영상 활용
  - 부모가 직접 대화하며 파악
  - 특별히 하고 있는 것이 없음 (항상 마지막 고정)
  - 기타 (직접 입력, 항상 맨 끝)
- **의도**: 경쟁 수단 파악 + "특별히 없음" 비율로 시장 공백 확인.
- **검증**: H3 — "특별히 없음" 30% 이상이면 가설 지지.

### Q8. 아이의 교육과 관련해 가장 아쉬운 점을 2개만 골라주세요 (최대 2개)
- **유형**: 체크박스 (최대 2개 선택 제한 — 구글폼 "응답 유효성 검사"로 설정)
- **필수**: O
- **선택지** (순서 랜덤):
  - 학원은 다니는데 아이가 진짜 좋아하는 건지 모르겠음
  - 성적 관리에 치여서 다른 걸 시도할 여유가 없음
  - 아이의 적성을 파악할 객관적 방법이 없음
  - 아이와 진로/흥미에 대해 깊이 대화하기 어려움
  - 좋은 체험/프로그램이 있어도 정보를 찾기 어려움
  - 딱히 아쉬운 점이 없음 (항상 마지막 고정)
- **의도**: 2개 제한이 핵심. 모든 게 아쉬울 순 있지만 "가장"을 고르게 함으로써 진짜 페인포인트가 드러남. 우리 제품과 직결되는 항목("적성 파악 방법 없음", "대화하기 어려움")이 상위에 오는지 확인.
- **검증**: H3 — "적성 파악 방법 없음" 또는 "대화하기 어려움"이 상위 2개 안에 드는 비율 40% 이상이면 가설 지지.
- **분석 팁**: 이 응답이 향후 랜딩페이지의 핵심 카피가 됨. 가장 많이 선택된 아쉬운 점 = 광고 헤드라인.

## 섹션 5: 솔루션 반응 (H4 검증)

> 섹션 설명: "마지막 섹션입니다. 아래 설명을 읽고 답해주세요."

### (설명 텍스트 — 질문 아님)
> "최근 AI를 활용한 새로운 교육 서비스가 등장하고 있습니다.
>
> 그 중 하나의 예시로, 아이가 관심 있는 주제에 대해 AI 전문가 캐릭터(예: 해양생물학자, 우주과학자, 요리사 등)와 대화하면서 자기만의 관심사를 탐색하고, 그 기록이 쌓여 부모가 아이의 '흥미 지도'를 확인할 수 있는 앱이 있다고 합니다."

### Q9. 주변 학부모 중에 이런 서비스에 관심을 가질 분이 얼마나 될 것 같으신가요?
- **유형**: 단일선택
- **필수**: O
- **선택지**:
  - 거의 없을 것 같다
  - 10명 중 1~2명 정도
  - 10명 중 3~4명 정도
  - 10명 중 5명 이상
  - 잘 모르겠다
- **의도**: 3인칭 프레이밍. "당신이 쓸 건가요?"는 체면 때문에 높게 나옴. "주변에 관심 가질 사람이 얼마나?"로 물으면 자기 투사(projection)가 일어나면서 더 솔직한 답이 나옴. 한국 학부모 대상 설문에서 특히 효과적.
- **검증**: H4 — "3~4명" + "5명 이상" 합계 40% 이상이면 가설 지지.

### Q10. 본인은 이런 서비스가 있다면 사용해보고 싶으신가요?
- **유형**: 단일선택
- **필수**: O
- **선택지**:
  - 무료라면 한번 써볼 것 같다
  - 유료(월 1만원 이내)라도 써볼 것 같다
  - 유료(월 1~3만원)라도 써볼 것 같다
  - 잘 모르겠다
  - 사용하지 않을 것 같다
- **의도**: 단순 의향이 아니라 가격 앵커를 넣어서 지불의향(WTP)까지 동시 측정. "무료라면"에만 몰리면 유료 전환이 어렵다는 신호. "유료라도"가 30% 이상이면 강한 PMF 신호.
- **검증**: H4 — "유료라도" 합계 30% 이상이면 가설 강하게 지지.

### Q11. 이런 서비스에 대해 가장 먼저 드는 생각은? (2개만 선택)
- **유형**: 체크박스 (최대 2개)
- **필수**: O
- **선택지** (순서 랜덤):
  - 아이가 재미있어할 것 같다
  - AI랑 대화하는 게 교육적으로 괜찮을지 모르겠다
  - 기존 적성검사보다 나을 것 같다
  - 화면 사용 시간이 늘어날까 걱정된다
  - 부모가 아이를 이해하는 데 도움이 될 것 같다
  - 실제 효과가 있을지 의문이다
  - 기타 (직접 입력)
- **의도**: 긍정·부정을 섞어서 자연스러운 반응 파악. 2개 제한으로 "가장 먼저 드는" 진짜 반응만 포착. 부정 반응의 구체적 내용 = 온보딩/마케팅에서 선제 해소할 포인트.
