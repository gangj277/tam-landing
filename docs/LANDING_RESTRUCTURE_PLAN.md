# 랜딩페이지 재구성 계획

> 작성일: 2026-04-01
> 목적: 14개 섹션 → 8개 섹션으로 압축. 학부모 후킹 + 핵심 밸류 전달 + 전환율 극대화.

---

## 핵심 원칙

1. **메커니즘이 아니라 결과를 판다** — "어떻게 작동하나"가 아니라 "우리 아이에 대해 뭘 알 수 있나"
2. **추상적 미래 불안이 아니라 구체적 현실을 보여준다** — "AI 시대가 온다"가 아니라 "이미 바뀌었다"
3. **모든 통계는 ★★★★★ 공식 출처** — 교육부, 통계청, OECD
4. **스크롤 4-5회 이내** — 부모가 스마트폰으로 읽는 상황 전제

---

### AS-IS (14개 섹션)
```

```

2. 교육 현실 — 고교학점제/사교육/결과 (신규)

```

## 섹션별 상세 설계

### 섹션 1: Hero

**목적:** 즉각적 공감 + 핵심 문제 제기

**카피:**
```
[섹션 라벨] 없음 (Hero이므로)

[헤드라인]
우리 아이, 뭘 좋아하는지 알고 있나요?

[서브카피]
중학생 10명 중 4명은 꿈이 없습니다.
이유의 절반은 '내가 뭘 좋아하는지 몰라서'.
매일 10분, 새로운 세계에서 아이가 자기 자신을 발견합니다.

[출처 — 작은 글씨]
교육부, 2024 초중등 진로교육 현황조사 (38,481명)

[CTA 버튼]
Primary: "무료로 시작하기" → /signup
Secondary: "1분 무료 진단" → /quiz
```

**변경 포인트:**
- 기존 "AI 시대, 아이들이 그 세계를 경험하고 있나요?" → **"뭘 좋아하는지 알고 있나요?"**로 교체
- 추상적("AI 시대") → 구체적("중학생 40%")
- 데이터가 즉시 등장하여 신뢰 확보

**통계:**
- 중학생 희망직업 없음 40% — 교육부 2024 ★★★★★
- 이유 1위 51.2% "뭘 좋아하는지 몰라서" — 교육부 2024 ★★★★★

### 섹션 2: 교육 현실 (신규)

**목적:** "지금 해야 하는 이유" = 구조적 긴급성

**카피:**
```
[섹션 라벨]
── 이미 바뀐 교육 현실

[헤드라인]
자기이해, 이제 선택이 아니라 필수입니다

[3개 카드]

카드 1: 고교학점제
  [숫자] 138개
  [설명] 2025년, 아이가 직접 과목을 골라야 합니다.
  하지만 68.1%가 적성이 아닌 입시 기준으로 선택합니다.
  [링크] 자세히 알아보기 → /guide/gogyohakjeomje
  [출처] 종로학원, 2025 (470명)

카드 2: 사교육의 방향
  [숫자] 44만원/월
  [설명] 10년 새 90% 올랐습니다.
  하지만 63%는 국영수. 아이가 자기를 발견하는 데 쓰이는 건?
  [링크] 더 읽기 → /guide/elementary-career-exploration
  [출처] 통계청, 2024 사교육비조사

카드 3: 그 끝에서
  [숫자] 10만 명/년
  [설명] 매년 10만 명의 대학생이 중도탈락합니다.
  "이 길이 내 길이 아니었다"고요.
  [링크] 더 읽기 → /guide/ai-era-career
  [출처] 교육부/대학알리미, 2024

[하단 한 줄]
공부는 세계 1등인데, 행복은 OECD 꼴찌입니다.
```

**통계:**
- 고교학점제 138개 과목 — 교육부 공식
- 68.1% 입시 기준 선택 — 종로학원 2025 (★★★☆, 방향은 확실)
- 44만원/월, 10년 +90% — 통계청 2024 ★★★★★
- 63% 국영수 — 통계청 2024 ★★★★★
- 10만 817명 중도탈락 — 교육부 2024 ★★★★★
- 공부 1등/행복 꼴찌 — OECD ★★★★★

**가이드 연결:** 각 카드에서 학부모 가이드로 트래픽 유도

### 섹션 3: 리프레이밍

**목적:** "지금까지의 방식 vs 새로운 방식" 관점 전환

**기존 유지 (소폭 수정):**
```
[인용문]
"아이는 안을 들여다본다고 자기를 이해하지 않습니다.
다양한 세계를 만나고 자기만의 방식으로 반응할 때,
비로소 자기 자신을 알게 됩니다."

[Before/After 비교표]
기존 방식                      탐의 방식
────────                     ────────
고정된 콘텐츠를 소비한다         매일 새로운 세계에 들어간다
AI가 답을 준다                 AI가 가능성을 열고, 아이가 고른다
결과가 똑같다                  선택에 따라 달라진다
끝나면 아무것도 남지 않는다      선택의 패턴이 쌓인다
```

**변경 포인트:**
- 내용은 기존과 동일, 위치만 변경 (Stats 뒤 → 교육현실 뒤)
- 교육현실 섹션에서 "제도가 바뀌었는데 준비가 안 됐다"를 본 후에 "그래서 다른 접근이 필요하다"로 연결

### 섹션 4: "이렇게 됩니다" (결과물)

**목적:** 제품의 핵심 아웃풋을 한 방에 보여줌

**기존 5개 섹션 통합:** SelfDiscovery + FirstSevenDays + Portfolio → 1개

```
[섹션 라벨]
── 자기 발견

[헤드라인]
놀고 있을 뿐인데, 아이가 보입니다

[서브카피]
성격검사 결과가 아닙니다.
매일의 선택에서 드러나는 진짜 아이의 모습입니다.

[주간 미러 목업 — 핵심 비주얼]

  이번 주 탐험한 세계
  ├ 화성 도시 시장
  ├ 동물 구조센터
  ├ 미래 학교 설계
  ├ 연구 탐사선
  └ 마을 분쟁 중재

  발견된 패턴
  · "완성도보다 독특함을 자주 선택합니다"
  · "다른 사람을 돕는 역할에 몰입도가 높습니다"
  · "여러 안을 비교한 뒤에 결정합니다"

  관심 지도 (7개 영역 바 차트)
  창작 & 표현        ████████░░ 82
  리더십 & 의사결정   ███████░░░ 70
  모험 & 상상        ██████░░░░ 68
  공감 & 관계        █████░░░░░ 58
  ...

[부모 인용 — 회색 배경 카드]
"아이가 이런 걸 중요하게 여기는지 처음 알았어요.
시험 점수로는 절대 볼 수 없는 것들이더라고요."

[하단 — 고교학점제 연결]
이 패턴을 바탕으로 고교학점제 과목도 추천됩니다:
사회문제탐구, 법과정치, 윤리와사상, 창업과경영
```

**변경 포인트:**
- 기존 SelfDiscovery 섹션의 핵심 비주얼(주간 미러 목업)을 메인으로
- FirstSevenDays의 "이런 게 보인다" 메시지 통합
- Portfolio의 고교학점제 과목 추천을 하단에 1줄로 포함
- Product Intro의 3가지 가치(경험/자기이해/AI리터러시)는 이 결과물이 자연스럽게 보여줌

### 섹션 5: 작동 방식

**목적:** "어떻게?" — 결과물을 봤으니 메커니즘을 간결하게

```
[섹션 라벨]
── 작동 방식

[헤드라인]
매일 10분, 이렇게 작동합니다

[3스텝 — 수평 또는 수직]
1. 만남
   AI가 매일 새로운 세계를 만듭니다.
   화성 도시, 법정, 연구소, 사회적 딜레마...

2. 선택
   아이가 역할·관점·해결책을 직접 고릅니다.
   정답이 없습니다. 아이의 선택이 곧 답입니다.

3. 발견
   선택의 패턴이 쌓입니다.
   시스템이 비춰주고, 가이드가 함께 이야기합니다.

[미션 예시 1개 — 가장 매력적인 것]
──────────────────────
"화성 도시의 시장이 되었습니다. 물이 부족합니다.
농장에 줄까요, 병원에 줄까요?"

→ 다음 날: 실제 2024 케냐 물 부족 사례.
  어제 내 선택과 현실을 비교합니다.
──────────────────────
```

**변경 포인트:**
- 기존 HowItWorks(4일 상세) + ExperienceShowcase(5카테고리) + AILiteracy(도구 3개) → **3줄 + 예시 1개**
- AI 리터러시는 별도 설명 안 함 — 미션 예시 안에서 "AI가 만든 세계에서 선택한다"가 자연스럽게 전달

### 섹션 6: 사람+AI (탐 가이드)

**목적:** AI만이 아닌 "사람+AI" 차별점 + 가격

**기존 유지 (소폭 압축):**
```
[섹션 라벨]
── 탐 가이드

[헤드라인]
AI만으로 끝나지 않습니다

[서브카피]
전문 가이드가 아이의 여정을 함께합니다.
앱의 경험을 현실의 대화로 연결하고, 성장을 부모님께 전합니다.

[3개 포인트]
1. 주 1회 가이드 세션
   월 1회 대면 + 3회 화상. 가르치는 게 아니라 아이의 생각을 탐색합니다.

2. 매일의 경험 동행
   가이드가 아이의 매일 10분을 앱에서 지켜봅니다. 아이는 혼자가 아닙니다.

3. 부모님 주간 리포트
   어떤 세계에 끌렸는지, 어떤 선택을 했는지. 시험 점수로는 볼 수 없는 것들.

[가격]
월 20만원
매일 AI 미션 + 주 1회 가이드 + 주간 부모 리포트 포함

[비교]
진로 상담사 1회 = 30~50만원.
탐은 매일 데이터를 쌓고, 매월 리포트를 만듭니다.
```

### 섹션 7: 신뢰

**목적:** 남은 우려 해소

```
[3개 항목 — 기존 4개에서 압축]

1. 안전한 AI
   다중 안전 필터로 부적절한 콘텐츠를 차단합니다.
   AI는 답을 주지 않습니다. 가능성만 열어줍니다.

2. 하루 10분
   하나의 경험은 10분. 끝없이 빠져드는 디자인이 아닙니다.
   짧고 강하게, 매일.

3. 평가하지 않습니다
   점수, 등급, 순위가 없습니다.
   선택을 판단하지 않고, 패턴만 비춰줍니다.

[파운더 1줄 — 기존 Founder Letter 대체]
"연세대 23학번, B2B AI 소프트웨어 2년 운영, CREAI+IT 창립.
20살에 느낀 문제를 10살부터 풀고 싶었습니다." — 강지민, 탐 TAM 대표
```

**변경 포인트:**
- 기존 4항목 → 3항목 (부모 리포트는 TamGuide에서 이미 커버)
- Founder Letter 전체 삭제 대신 **1줄 요약**으로 신뢰 섹션에 통합
- 파운더의 풀 스토리는 추후 /about 페이지로 분리 가능

 결국 아이는 자기를 모른 채 선택해야 하네"   → 긴급성 + 불안

"해볼까?"                               → 전환
```
