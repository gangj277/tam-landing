## 컨택 포인트
- **담당 부서**: 연구기획센터 / 국가진로교육연구센터

- **이메일**: 공개 개인 이메일 미확인
- **전화**: 044-415-5228 / 044-415-5000, 5100
- **기타**: `연구과제 제안` 페이지 연중 상시 운영

## 확인된 포인트
- 공개 개인 메일은 없지만 연구협력 창구는 분명하다.
- TAM은 `진로탐색 종단 데이터`, `현장 실험`, `체험 중심 진로교육 효과`라는 연구 질문과 잘 맞는다.
- 메시지는 제품 소개보다 연구 협력 제안서 성격이 적합하다.

## 우리가 원하는 것
- 연구과제 제안 방향성 확인
- 공동 연구 또는 효과성 검토 가능성 확인
- 적절한 센터/담당 연결
