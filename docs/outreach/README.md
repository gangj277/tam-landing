## 메시징 원칙
- 정부/공공: `체험 중심 진로교육`, `정책 정합성`, `실증/효과 검증`
- 비영리/재단: `교육격차 해소`, `청소년 임팩트`, `측정 가능한 결과`
- 기업 CSR: `기존 프로그램의 사전/사후 탐색 도구`, `참여 학생 경험 강화`, `짧은 파일럿`
- 협회/학회: `학술적 가치`, `종단 데이터`, `회원 네트워크 확산`
- 민간 플랫폼/기관: `등록`, `도입`, `오프라인 체험과 디지털 연계`
- 청소년 활동/복지: `방과후 진로 루틴`, `돌봄 아동 진로 기회`, `일상적 탐색`
- 교육청/센터: `자유학기 연계`, `진로상담 보완`, `학교 단위 파일럿`
- 행사/전시: `생태계 노출`, `교사/관계자 접점`, `차별화된 카테고리`
- 학원/코칭: `학습 성향 + 진로 성향`, `학습 동기의 근원`, `기존 체계에 붙는 모듈`
- 진로체험 업체: `오프라인 체험 전후 디지털 연계`, `공동 패키지`, `학교 번들 납품`

## 사용 방법
- 각 기관 파일의 `## 바로 발송용`만 보면 된다.
- 이메일 기관은 `To`, `Subject`, `본문`을 그대로 복사해 발송하면 된다.
- 문의폼 기관은 `문의폼 입력값`의 `제목`과 `내용`을 그대로 붙여넣으면 된다.
- 공개 이메일이 없는 기관은 `전화 후 바로 보낼 메일 본문`을 기준으로 담당자 확인 뒤 발송하면 된다.

## 공통 소개 문구
> TAM은 초등학생부터 중학생까지를 위한 AI 기반 진로탐색 루틴입니다.  
> 아이가 매일 5분 시나리오 체험과 AI 대화를 통해 다양한 세계를 경험하고, 그 과정에서 드러나는 관심과 사고 패턴을 축적합니다.  
> 진로적성검사처럼 한 번 분류하는 도구가 아니라, 체험 중심 진로교육을 일상 루틴으로 만드는 앱입니다.

## 카테고리별 CTA
- 정부 지원: 실증 가능 여부, 적합한 접수 경로, 사업 적합성 검토
- 교육 플랫폼: 등록 절차, 도입 가능 범주, 소개 자료 제출 방법
- 진로교육 기관: 공동 파일럿, 사전/사후 탐색 도구 연계
- 협회/학회: 연구/발표/회원 연계, 효과성 검토
- CSR: 기존 프로그램 참여자 대상 2주 파일럿
- 벤치마크: 운영자 대화, 학습, 시장/도입 구조 파악
- 청소년 활동/복지: 방과후 진로 프로그램 연계, 모델 센터 파일럿
- 비영리/재단: 교육격차 대상 무상 파일럿, 임팩트 공동 측정
- 교육청/센터: 자유학기/상담 연계 경로 확인, 담당 장학사 연결
- 행사/전시: 참가 비용/부스 확인, 스타트업 세션 참여
