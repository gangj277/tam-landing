# 초록우산 어린이재단

## 기본 정보
- **유형**: 비영리 / 아동복지 전문기관
- **웹사이트**: https://www.childfund.or.kr
- **주소**: 서울시 중구 무교로 20 어린이재단빌딩
- **TAM과의 관계**: 교육격차에 놓인 아동의 진로탐색 기회를 임팩트 언어로 설계할 수 있는 대형 비영리 파트너

## 컨택 포인트
- **담당 부서**: 기업제휴 / 사회공헌 협력
- **담당자명**: 미공개
- **이메일**: 공개 파트너십 이메일 미확인
- **전화**: 1588-1940
- **기타**: 교육부 인정 교육기부 우수기관

## 확인된 포인트
- 돌봄, 자립, 교육, 건강, 주거 5개 영역에서 아동복지 사업을 운영하고 있다.
- TAM은 교육 영역 중 진로탐색에 특화된 도구로, 재단의 교육격차 해소 서사와 맞닿는다.
- 대형 재단이므로 첫 접점은 기업제휴보다 `교육 프로그램 연계 소규모 파일럿`이 현실적이다.
- 임팩트 측정 가능한 형태(참여율, 관심 변화, 보호자 반응)로 제안해야 한다.

## 우리가 원하는 것
- 교육 프로그램 내 진로탐색 도구 연계 파일럿
- 교육격차 대상 아동에게 TAM 무상 제공 모델 검토
- 재단형 임팩트 파일럿 설계
