# 드림캐쳐 에듀투어

- **유형**: 민간 / 찾아가는 진로체험 전문업체
- **웹사이트**: https://dcedutour.co.kr
- **주소**: 미확인

## 확인된 포인트
- 전현직 전문가가 학교로 직접 찾아가 초등/중고등 대상 진로체험을 제공한다.
- 50여 개 직업 분야의 전문 강사를 보유하고 있다.
- 학교, 청소년수련관, 방과후수업, 돌봄교실 등 다양한 기관에 납품한다.
- TAM은 이 업체의 프로그램 전후에 붙는 디지털 도구로 포지셔닝할 수 있다 — 체험 전 관심 탐색, 체험 후 성찰 리포트.
- 파트너십 문의폼이 공식 창구다.

## 우리가 원하는 것
- 드림캐쳐 프로그램 전후에 TAM 연계 가능성 확인
- 공동 패키지 (오프라인 체험 + 디지털 탐색) 설계
- 학교 납품 시 TAM 번들 제안
