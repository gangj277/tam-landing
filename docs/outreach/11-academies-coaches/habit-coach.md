# 한국습관교육센터

## 기본 정보
- **유형**: 민간 / 습관코칭·인성교육 전문
- **웹사이트**: https://www.habit-coach.com
- **주소**: 미확인
- **TAM과의 관계**: 학교/기관 대상 습관 변화 교육을 제공하며, '매일 5분 진로탐색 습관'이라는 TAM의 포지셔닝과 직접 맞닿음.

## 컨택 포인트
- **담당 부서**: 교육사업
- **담당자명**: 미공개
- **이메일**: 공개 이메일 미확인
- **전화**: 미확인 (웹사이트 문의폼)
- **기타**: 2010년부터 학교·기업 대상 습관 콘텐츠 연구, 강의, 코칭 진행

## 확인된 포인트
- 전문 코치들이 학교와 기업에 방문해 습관 변화 교육을 진행한다.
- 중고등학생 습관변화교육 프로그램을 교육모듈 형태로 운영한다.
- '습관'이라는 키워드가 TAM의 '매일 5분 루틴'과 정확히 겹친다.
- TAM을 '진로탐색 습관을 만드는 도구'로 포지셔닝하면 기존 프로그램에 자연스럽게 붙는다.

## 우리가 원하는 것
- 습관교육 프로그램에 '진로탐색 습관' 모듈로 TAM 연계
- 학교 출강 시 TAM 번들 가능성 확인
- 공동 프로그램 설계
