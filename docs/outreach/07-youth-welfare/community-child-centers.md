## 확인된 포인트
- 지역아동센터는 전국 4,000개 이상으로, 주로 교육격차에 놓인 아동을 대상으로 방과후 보호·교육·문화를 제공한다.
- 기존 프로그램에 학습지도, 예체능, 문화체험이 있지만 `일상적 진로탐색 루틴`은 사실상 부재하다.
- TAM은 교육격차 해소 서사와 가장 강하게 맞닿는 채널이다.
- 중앙지원단을 통해 모델 센터 1~2곳 파일럿 후 확산하는 구조가 현실적이다.

## 우리가 원하는 것
- 지역아동센터 프로그램에 TAM 진로탐색 루틴 연계 가능성 확인
- 모델 센터 1~2곳 파일럿 기회
- 교육격차 해소 임팩트 데이터 공동 확보
