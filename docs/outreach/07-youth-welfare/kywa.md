# 한국청소년활동진흥원 KYWA

## 기본 정보
- **유형**: 준정부기관 / 청소년활동 지원
- **웹사이트**: https://www.kywa.or.kr
- **주소**: 서울특별시 서대문구 경기대로 47
- **TAM과의 관계**: 청소년방과후아카데미 전국 400+개소의 진로 프로그램에 디지털 탐색 루틴을 연결할 수 있는 최대 규모 채널

## 컨택 포인트
- **담당 부서**: 정책사업부 (청소년방과후아카데미 담당)
- **담당자명**: 신지혜
- **이메일**: 공개 파트너십 이메일 미확인
- **전화**: 02-330-2894 (방과후아카데미), 02-330-2800 (대표)
- **기타**: e청소년 포털 https://www.youth.go.kr/yaca

## 확인된 포인트
- 청소년방과후아카데미는 전국 400개 이상 운영기관에서 초4~중3 대상 방과후 돌봄+활동을 제공한다.
- 프로그램에 `진로개발역량` 영역이 명시돼 있어 TAM과 직접 연결된다.
- TAM은 기존 진로체험 프로그램의 사전/사후 루틴으로 붙이는 형태가 가장 자연스럽다.
- 규모가 크므로 첫 접점은 본부 파일럿 제안보다 `1~2개 운영기관 단위 실험` 제안이 현실적이다.

## 우리가 원하는 것
- 방과후아카데미 진로 프로그램과 TAM 연계 가능성 확인
- 1~2개 운영기관 단위 소규모 파일럿 기회
- 적절한 담당 부서/운영기관 연결
